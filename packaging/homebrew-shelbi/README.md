# Shelbi Homebrew Tap Bootstrap

This directory is the seed content for the dedicated Homebrew tap repository.
Create the external repository as:

```bash
gh repo create jlong/homebrew-shelbi --public --description "Homebrew tap for Shelbi"
```

Then copy the contents of this directory into that repository. Homebrew maps
`jlong/homebrew-shelbi` to the user-facing tap name `jlong/shelbi`, so the
install command is:

```bash
brew tap jlong/shelbi
brew install shelbi
```

The first real `Formula/shelbi.rb` is generated by the release workflow after
GitHub Release artifacts and `checksums.txt` exist. Generate it manually with:

```bash
ruby scripts/release/update-homebrew-formula.rb \
  --tap-dir /path/to/homebrew-shelbi \
  --version 0.1.0 \
  --checksums /path/to/checksums.txt \
  --github-repo jlong/shelbi
```

The generator always requires `shelbi_Darwin_arm64.tar.gz` and includes
`shelbi_Darwin_x86_64.tar.gz` only when that archive is present in
`checksums.txt`.

If ownership moves from `jlong/*` to a `shelbi/*` organization, create
`shelbi/homebrew-shelbi`, set the main repository variable
`HOMEBREW_TAP_REPOSITORY=shelbi/homebrew-shelbi`, and publish the install
command as:

```bash
brew tap shelbi/shelbi
brew install shelbi
```
